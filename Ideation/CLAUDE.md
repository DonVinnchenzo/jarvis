# Ideation — CLAUDE.md

## Purpose

This folder holds research, market analysis, and pre-spec exploration for Jarvis features.

## Workflow

1. **Research** — Save findings to `Ideation/[FEATURE]-RESEARCH.md`
2. **Write spec** — Create `specs/NNN-feature-name.md` using the template
3. **Planning** — Save implementation plan to `Ideation/[FEATURE]-IMPLEMENTATION-PLAN.md`
4. **Build** — Implement per plan
5. **Verify** — Run `/spec-verify`

## Rules

- Every idea starts in `BACKLOG.md` before getting a spec
- Research docs capture alternatives considered and decisions made
- Don't over-research — timebox to what's needed for a spec
